# TabBar 图标文件说明

此目录需要放置以下图标文件（PNG格式，建议尺寸 81x81 像素）：

## 需要的图标列表

1. **tab-home.png** / **tab-home-active.png** - 首页图标
2. **tab-record.png** / **tab-record-active.png** - 记录图标
3. **tab-nutrition.png** / **tab-nutrition-active.png** - 营养图标
4. **tab-wheel.png** / **tab-wheel-active.png** - 决策图标
5. **tab-profile.png** / **tab-profile-active.png** - 个人中心图标

## 图标设计建议

- 使用温馨的粉色系（#F8A5B6）和浅绿色系（#A8D8B9）
- 图标风格保持简洁、圆润，符合孕期主题
- 普通状态使用灰色系，选中状态使用粉色
- 建议使用线性图标风格

## 注意事项

- 图标尺寸建议：81x81 像素
- 格式：PNG（支持透明背景）
- 命名必须与上述列表完全一致
